終極 CSS 偽類展示

基本狀態偽類

:hover（滑鼠懸停）

.hover-demo:hover {
    background-color: yellow;
}

:active（點擊時）

.active-demo:active {
    background-color: red;
}

:focus（獲得焦點）

.focus-demo:focus {
    outline: 3px solid blue;
}

空元素 :empty

有內容的 div

空的 div 被隱藏了

div:empty {
    display: none;
}

否定選擇器 :not()

這是普通段落

這是特殊段落

這也是普通段落

p:not(.special) {
    color: gray;
}

目標元素 :target

點擊這裡
我是目標元素（點擊上面的連結看效果） 
:target {
    background: yellow;
}

表單相關偽類

:disabled（禁用狀態）

input:disabled {
    background-color: #f0f0f0;
}

:checked（選中狀態）

input:checked + label {
    color: green;
}

表單驗證偽類

input:valid {
    border-color: green;
}

input:invalid {
    border-color: red;
}

input:required {
    background-color: #fff3f3;
}

結構性偽類

  • 第一項 (:first-child)
  • 第二項 (even)
  • 第三項
  • 第四項 (even)
  • 第五項
  • 最後一項 (:last-child)
li:first-child {
    font-weight: bold;
}

li:last-child {
    border-bottom: none;
}

li:nth-child(even) {
    background: #f0f0f0;
}

li:nth-child(odd) {
    background: #ffffff;
}

連結狀態偽類

未訪問的連結
已訪問的連結 
a:link {
    color: blue;
}

a:visited {
    color: purple;
}

a:hover {
    text-decoration: none;
}

選擇偽類 ::selection

試試選取這段文字！文字被選中時會變成黃底黑字。

.selection-demo::selection {
    background-color: yellow;
    color: black;
}
